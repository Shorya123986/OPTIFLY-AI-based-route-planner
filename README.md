# OPTIFLY-AI-based-route-planner

## 📌 Overview

OptiFly is an intelligent flight route optimization system developed using the A* Search Algorithm.

The application allows users to find the optimal flight path between two airports while optimizing one of the following:

- Distance
- Fuel Consumption
- Travel Time

The project visualizes the generated route on a map using NetworkX and Matplotlib.

---

## Features

✔ A* Search Algorithm

✔ Interactive GUI using Tkinter

✔ Route Optimization based on:
- Distance
- Fuel
- Time

✔ Airport Network Visualization

✔ Displays Optimal Path

✔ Calculates Total Fuel Consumption

✔ Uses Real Airport Coordinates
